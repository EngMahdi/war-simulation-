## 🇬🇧 English Description

A browser-based interactive war simulation tool — no installation required.
Analyzes the military balance of power between two opposing coalitions
using fully adjustable variables with real-time result updates.
No hardcoded conclusions — every assumption is editable by the user.

### 📊 Ten Simulation Categories

- Military Hard Power
- Air Superiority
- Air Defense Systems
- Ballistic & Cruise Missiles
- Naval Control & Strategic Straits
- Geography & Logistics
- Industrial War Capacity
- Regional Allies & Proxy Forces
- Ideological & Societal Factors
- Leadership Stability

### ⚙️ Features

- Real-time calculation — results update instantly on every change
- Fully editable variables — every score adjustable from 0 to 100
- Impact ratio system — integer weights from 1 to 99 per category
- Weight Manager — presets for Blitz War, Attrition, Proxy War, Naval War
- Auto-normalization — weights automatically balanced to 100%
- Undo history — up to 30 steps back
- Radar chart — visual capability distribution
- Category overview — comparative bars across all dimensions
- Add / Delete custom categories and variables
- AI Prompt Export — short, medium, or detailed analytical prompt
- Save & Load model as JSON
- Print / PDF export
- Fully responsive — works on mobile and desktop
